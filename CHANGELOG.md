# Changelog

All notable changes to the Lumina project will be documented in this file.

## [Unreleased]

### Added
- Power BI-like LumaDash feature - Full conversational dashboard with natural language chart creation
- 9+ advanced chart types (bar, line, pie, doughnut, area, map, gauge, funnel, KPI, tables)
- Natural language command system for dashboard control (/add, /remove, /update, /theme, etc.)
- Advanced dashboard customization (color themes, title colors, layouts, time filters)
- Collaboration features (save, export, share, real-time collaboration)
- Advanced analytics integration (anomaly detection, trend analysis, correlation analysis)
- Interactive dashboard preview mode with edit/preview toggle
- Chat command integration for dashboard control
- Dashboard command processor for natural language interpretation
- Comprehensive documentation for Power BI-like features

### Changed
- Enhanced AdvancedLumaDash component with Power BI-like capabilities
- Improved theme toggle functionality to work with data-theme attribute
- Enhanced UI with 3D effects, animations, and consistent styling
- Updated ChartWrapper component to properly handle chart data
- Refactored AppPage to include view mode toggle
- Replaced basic LumaDash with AdvancedLumaDash component

### Fixed
- Theme toggle not working properly in dark/light mode
- TypeScript error with ChartWrapper component props
- Various UI/UX improvements

## [1.0.0] - 2025-09-25

### Added
- Initial release of Lumina platform
- Natural language data analysis capabilities
- Chat interface for conversational data exploration
- File upload and analysis functionality
- AI-powered insights generation
- Dark/light theme support
- Responsive design for all device sizes
- LumaDash feature - Conversational dashboard with natural language chart creation
- Toggle between chat interface and dashboard view
