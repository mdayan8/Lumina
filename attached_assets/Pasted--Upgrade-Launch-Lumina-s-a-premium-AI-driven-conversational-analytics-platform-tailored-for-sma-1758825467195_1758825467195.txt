Lumina s a premium, AI-driven conversational analytics platform tailored for small and medium-sized businesses (SMBs). A data pipeline is the flow of data from its source (here, CSV files) to the final actionable output (insights and charts). For Lumina’s MVP:

CSV Upload → Parsing → Schema Inference → AI Analysis → Chart Generation → Insight Display

2. Step-by-Step Flow

Step 1: CSV Upload

User drags-and-drops or selects a CSV file.

Frontend: React component (react-dropzone) handles file selection.

File sent via API to backend storage (AWS S3).

Metadata logged in PostgreSQL: filename, rows, columns, upload timestamp.

Step 2: Data Parsing & Schema Inference

Backend service (Python FastAPI) reads CSV using pandas.

Tasks performed:

Detect columns and data types (numeric, text, date).

Identify key metrics (e.g., sales, quantity, dates).

Handle missing or inconsistent values (drop/fill).

Output: structured dataframe ready for analysis.

Step 3: AI Prompt Construction

User sends a query through the chat interface, e.g., “Top 5 customers last month?”

Backend constructs a prompt for GPT-4 Turbo:

Includes data summary: column names, sample rows, numeric ranges.

Includes user query in natural language.

GPT-4 Turbo generates:

Analysis in plain English.

Suggested actions or insights.

Step 4: Chart Generation

Pandas + Chart.js backend logic prepares visual representation:

Bar charts: Sales per product/customer

Line charts: Trends over time

Pie charts: Cost breakdown, market share

Chart data returned to frontend as JSON, rendered using react-chartjs-2.

Core Charts for CSV Analysis MVP:

Top N Customers/Products – bar chart

Sales/Revenue Trends – line chart

Category/Cost Breakdown – pie chart

Forecast (optional MVP) – simple line trend

Step 5: Insight Display in Chat

Frontend displays:

Plain-English answer (from AI)

Embedded chart (bar/line/pie)

Optional “Why this matters?” tooltip

Download button for PDF report

User can ask follow-up questions, creating a conversational data exploration experience.
